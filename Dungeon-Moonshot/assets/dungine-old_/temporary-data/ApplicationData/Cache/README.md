## /ApplicationData/Cache/

This is where an application's cached data should be stored.

Cached data *is* persistant, and *is not* removed when an application exits.

Data that *is removed* when an application exits should be stored in **/ApplicationData/Tmp/**.

### Folder structure

There is not a specific folder structure for **/ApplicationData/Cache/**.
